# Project Overview - Daily Spending PWA

## Project Name
DailySpend PWA (temporary)

## Description
DailySpend is a lightweight offline-first Progressive Web App (PWA) that helps users track their total spending per day, compare spending across months, manage daily budget limits, and track upcoming bills.

The application is designed to work fully offline and store all data locally in the user's device.

## Goals
- Allow user to record total spending per day quickly.
- Provide monthly summaries and spending analytics.
- Support daily budget limit with warnings.
- Track upcoming bills and due reminders.
- Provide optional AI forecasting and spending advice.

## Target Users
- Students tracking daily expenses quickly.
- Users who do not want complex transaction entry.
- Minimalist budget tracking users.

## Core Features
### Daily Spending Input
- User enters total spending for the day (RM).
- Optional note field.
- Edit/delete daily record.

### Budget Limit
- User sets daily budget limit.
- App shows remaining budget.

### Monthly Summary
- Total spent per month.
- Average spending per day.
- Highest spending day.
- Comparison with previous month.

### Upcoming Bills
- User adds bills with due dates.
- Recurring bills supported.
- Reminder notifications.

### AI Support (Optional)
- Forecast monthly spending based on daily trend.
- Recommend safe daily spending.
- Identify risky spending patterns.
